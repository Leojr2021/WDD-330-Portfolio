Ajax

The meaning is Asynchronous JavaScript and XML, these are the conbination of HTML or XHTML, CSS, JavaScript, DOM, XML, XSLT, and most importantly the XMLHttpRequest. With these conbination we can develop faster web apps and increase the perfomance and inprove UX.

Some of the core functionality are Fetch API, Server-sent events, XML.

Axios

This tool is very used by developer, it's a promise based HTTP client, which uses XMLHttpRequest internally.
